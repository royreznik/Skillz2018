Java Documentation

class Asteroid

package pirates

extends SpaceObject › GameObject › MapObject › SkillzObject

A big obstacle that can be pushed across the map, and respawns every time it's pushed out of the map.

The most important API is:
  • direction
  • speed

Variables

direction:Location

The direction of the asteroid's movement. The asteroid's location next turn is asteroid.location + asteroid.direction

initialLocation:Location

The Asteroid's initial location.

spawnTurns:int

The turns it takes for the Asteroid to respawn.

speed:int

The speed the asteroid moves every turn.

turnsToRevive:int

The number of turns left for the Asteroid to revive. 0 if it's alive.
