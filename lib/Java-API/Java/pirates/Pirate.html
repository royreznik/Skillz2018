Java Documentation

class Pirate

package pirates

extends SpaceObject › GameObject › MapObject › SkillzObject

This class represents a pirate. Pirates are the main actors controlled by the players.

The most important API is:
  • capsule
  • canPush
  • hasCapsule
  • push

Variables

capsule:Capsule

The Capsule that is carried by this Pirate. If the Pirate has no Capsule, this is equal to null.

initialLocation:Location

The Pirate's initial location.

maxSpeed:int

The maximum speed of the Pirate (maximum number of steps it can go in a single turn).

numPushesForCapsuleLoss:int

The number of pirates that need to push a Pirate with a Capsule so that the Pirate will drop it.

pushDistance:int

The distance the Pirate's push attack pushes objects.

pushRange:int

The push range of the Pirate.

pushReloadTurns:int

The number of turns left until the Pirate can push again. If the Pirate can push, this is equal to 0.

spawnTurns:int

The turns it takes for the Pirate to respawn.

turnsToRevive:int

The number of turns left for the Pirate to revive. 0 if it's alive.

canPush(spaceObject:SpaceObject) : boolean

Returns whether or not this Pirate can push the given space object, checking the distance and size as well as reload turns. If this function returns true, the Pirate will succeed to push the space object.
