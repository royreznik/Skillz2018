Java Documentation

class PirateGame

package pirates

extends SkillzObject

This class represents the current state of the game, and is passed in the beginning of each turn to the player's doTurn function.

The most important API is:
  • getEnemyCapsules
  • getEnemyLivingPirates
  • getEnemyMotherships
  • getMyCapsules
  • getMyLivingPirates
  • getMyMotherships
