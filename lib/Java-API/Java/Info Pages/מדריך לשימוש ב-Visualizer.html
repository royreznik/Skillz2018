info-file מדריך לשימוש ב-Visualizer

<p>את המשחקים בתחרות ניתן לראות רק לאחר שהם הורצו בהיצג הגרפי של המשחק - ה Visualizer.</p> <p>ה Visualizer מגיע עם מגוון כלים והגדרות כדי להקל על הבנת המשחק וצפיתו.</p> <p>בעזרת הכפתורים בתחתית התפריט בצד שמאל, ניתן לשלוט בתצוגה של המשחק.</p> <p>ניתן לשנות את מהירות הרצה, התור נוכחי וגם לזוז אחורה וקדימה בעזרת הגלגלת כאשר העכבר נמצא על מפת המשחק.</p> <p><img alt="Simulator buttons 1" src="../img/Visualizer/Controls.png"></p> <p>לחיצה על Open Settings תפתח את תפריט ההגדרות, שם ניתן לכבות ולהדליק את המוזיקה ואת הסאונד, וכן לשנות אפשרויות ויזואליות שונות.</p> <p><img alt="Menu" src="../img/Visualizer/Settings.png"></p> <p>לחיצה על Keyboard Shortcuts תפתח את תפריט קיצורי המקשים. שם נמצאים קיצורי מקשים נוחים המאפשרים שליטה בתור ובתפריטים הפתוחים.</p> <p><img alt="Menu" src="../img/Visualizer/Shortcuts.png"></p> <p>לחיצה על שם השחקן תפתח את חלון ה- Debug, בו ניתן לראות את ההודעות שהבוט מדפיס.</p> <p>מכיוון שהמשחק הורץ לפני הצפייה בו, הודעות אלו הם הדרך שלכם להבין מדוע הבוט שלכם עושה מה שהוא עושה.</p> <p>להודעות אלו יש כמה מגבלות:</p> <ol> <li>עד כ 1,000 שורות בכל תור מסוים.</li> <li>עד כ 1,000,000 תווים לאורך משחק שלם.</li> </ol> <p>לאחר שבוט עובר את אחת המגבלות האלו לא יוצגו ממנו הודעות דיבאג נוספות במשחק זה.</p> <p><img alt="Debug messages" src="../img/Visualizer/Debug.png"></p> <p>עמידה עם הסמן של העכבר על משבצת במפה תציג את הנתונים הבאים בצדו השמאלי העליון של המסך:</p> <ol> <li>מיקום המשבצת (שורה ועמודה).</li> <li>עבור כל האובייקטים הקרובים לנקודה, את המיקום שלהם, את המזהה הייחודי שלהם ואת השחקן אליו הם שייכים.</li> </ol> <p><img alt="Hint" src="../img/Visualizer/Hint.png"></p>
