<h1>התחברות והרשמה</h1> <p>בכניסה הראשונה לאתר, הרשמו בעזרת הפעולות הבאות:</p> <p><img alt="Log in" src="../img/UserGuide/LogIn.png"></p> <p>בחרו בהזדהות משרד החינוך, והתחברו דרכם. לאחר ההתחברות תגיעו לעמוד ההרשמה של skillz 2018:</p> <p><img alt="Sign up" src="../img/UserGuide/SignUp.png"></p> <p>מלאו את פרטיכם האישיים, נלחץ "הרשם" וישלח אליכם מייל לאישור. לאחר שתאשרו את המייל ותתחברו, תגיעו לדף הבא:</p> <p><img alt="Ethic Code" src="../img/UserGuide/EthicCode.png"></p> <p>בתחתית העמוד יהיה עליכם לאשר את הקוד האתי. לאחר שקראתם אותו תגיעו לעמוד הטורנירים. בנוסף, מימין יש תפריט בו תוכלו לראות / לשנות את הפרופיל האישי שלכם. </p> <h1>היכרות עם דפי האתר</h1> <h3>דף הטורנירים</h3> <p>בחלק הטורנירים תוכלו להתחבר לטורנירים אליהם אתם מוזמנים. על מנת לעשות זאת, לחצו על "הזמנות לטורנירים", ובחרו את הטורניר אליו הוזמנתם.</p> <p><img alt="Tournaments" src="../img/UserGuide/Tournaments.png"></p> <p>לאחר מכן, תגיעו לעמוד "נהל קבוצה". בעמוד זה תוכלו לשנות את שם הקבוצה ולבחור אווטאר וסמל.</p> <p><img alt="Group Page" src="../img/UserGuide/GroupPage.png"></p> <p>בתפריט מימין נפתחו הדפים הבאים:</p> <h3>העלאת קוד</h3> <p>בדף זה ניתן להעלות גרסאות קוד חדשות ולהוריד את הגרסאות הקודמות לפי זמן. הגרסאות שניתן להוריד הן רק גרסאות שהועלו דרך דף זה או על ידי submit בדף ה- webkit.</p> <p><img alt="Code" src="../img/UserGuide/Code.png"></p> <h3>תוצאות</h3> <p>בדף זה ניתן לראות את מיקום הקבוצות, ניקוד הקבוצות בסבבים השונים והניקוד הכללי. </p> <p><img alt="Results" src="../img/UserGuide/results.png"></p> <h3>הוראות</h3> <p>בדף זה מתוארים בקצרה מטרות השחקן ואופן המשחק. </p> <p><img alt="Instructions" src="../img/UserGuide/Instructions.png"></p> <h3>מדריכים</h3> <p>בדף זה תוכלו למצוא את כל המדריכים הנדרשים על מנת לכתוב בוט למשחק, בסיסי ומורכב כאחד. המדריכים המופיעים בראש העמוד הם כלליים ועוסקים בעיקר בחוקי המשחק. בעמוד זה תוכלו למצוא מדריכים מפורטים לכל אחת מן השפות הניתנות לשימוש בתחרות- python, c#, java, javascript</p> <p><img alt="Documentation" src="../img/UserGuide/Docs.png"></p> <h3>קבוצות</h3> <p>בדף זה תוכלו לראות את שאר הקבוצות בטורניר ואת חברי הקבוצה.</p> <p><img alt="Groups" src="../img/UserGuide/Groups.png"></p> <h3>ה- webkit</h3> <p>דף זה נראה כך והוא מחולק לארבעה חלקים: </p> <p><img alt="webkit" src="../img/UserGuide/Webkit.png"></p> <ul> <li> <p>בצד ימין למעלה נמצא עורך הקוד שלכם - בתוכו תוכלו לכתוב את הקוד של הבוט שלכם. </p> </li> <li> <p>בצד ימין למטה נמצא מנהל הבוט שלכם - בו ניתן לשחק מול בוטים אחרים ברמות שונות ולנהל את גרסאות הקוד שלכם. בנוסף בחלק זה ישנם ארבעה כפתורים: </p> <ul> <li> <p>submit - יעדכן את הגרסה שלכם מול השרתים והיא תחשב כגרסה העדכנית ביותר.</p> </li> <li> <p>retrieve - יעדכן אתכם אל מול הגרסה העדכנית ביותר של הבוט שלכם (זאת שעשיתם לה submit).</p> </li> <li> <p>save - ישמור את הקוד שלכם <strong>רק</strong> במחשב שלכם עד אשר תרעננו / תשנו דף.</p> </li> <li> <p>load - יעדכן אתכם אל מול הכרסה אליה עשיתם save לאחרונה.</p> </li> </ul> </li> <li> <p>בחלקו השמאלי העליון של הדף נמצא ה- visualizer, בו ניתן לצפות ב- replay של המשחק, לשלוט על קצב ההרצה ועוד.</p> </li> <li> <p>בצד שמאל למטה נמצא ה- logger, המציג את הדפסות השחקן במהלך המשחק.</p> </li> </ul>
