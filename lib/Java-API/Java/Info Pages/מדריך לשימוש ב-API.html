info-file מדריך לשימוש ב-API

<p>ממשק תכנות יישומים, המוכר בשמו הלועזי (API (Application Programming Interface הוא ערכה של פעולות ואובייקטים המוצעים לשימוש בכתיבת תוכנה. דוגמאות קלאסיות ל-API היא פונקציית ההדפסה <span dir="ltr">System.out.println()</span>, או הפעולות ב- Math. פעולות אלו הן פעולות בהן השתמשתם פעמים רבות בעבר מבלי שנדרשתם לכתוב אותן בעצמכם.</p> <p>ה- API של המשחק כולל בתוכו את כל הפעולות ההכרחיות לצורך כתיבת בוט, וכן מספר פעולות עזר שיוכלו לסייע לכתיבת הבוט שלכם. לדוגמא, לכל האובייקטים במשחק יש פונקציית inRange (השם שונה בכל שפה, בהתאם למוסכמות של השפה) ופונקציית distance, כדי שלא תצטרכו לממש פונקציות כאלו בעצמכם.</p> <p>לכל שפה יש מוסכמות משלה עבור מבנה ה- API ושמות הדברים בו. בכל השפות בתחרות שמות המחלקות נכתבים ב- UpperCamelCase, לדוגמה PirateGame או MapObject. ההבדל בין השפות בא לידי ביטוי בשמות הפונקציות והמשתנים:</p> <ul> <li>ב- Python פונקציות ומשתנים נכתבים ב- snake_case (למשל- in_range).</li> <li>ב- <span dir="ltr">C#</span> פונקציות פומביות נכתבות ב- UpperCamelCase (למשל- InRange).</li> <li>ב- Java וב- JavaScript פונקציות ומשתנים נכתבים ב- lowerCamelCase (למשל-inRange).</li> </ul> <p>ה- API של כל השפות זהה פרט להבדלי השמות הנובעים מהמוסכמות השונות בשפות השונות.</p> <p>לשם נוחות, ה-API מחולק למחלקות וקבצים. מרבית ה- API הרלוונטי עבורכם נמצא במחלקה Pirate, המייצגת את הפיראטים בהם אתם שולטים, ובמחלקה PirateGame, האובייקט המכיל את מצב המשחק הנוכחי, ומסופק לבוט שלכם בתחילת כל תור (זה, למעשה, ה- game שאתם מקבלים כפרמטר לפונקציית ה- doTurn שלכם).</p> <h2>הדף הראשי - index.html</h2> <p>בדף זה מופיעים עץ ירושה ורשימה של כל המחלקות הקיימות במשחק. לחיצה על שם המחלקה בעץ או ברשימה תפתח רשימה של כל הפונקציות והמשתנים השייכים לאותה מחלקה, יחד עם תיאור קצר.</p> <p><img alt="index" src="../img/Api/index.png"></p> <p>נוסף על הדף הראשי, לכל מחלקה קיים דף משלה, המציג את המידע המלא על אודות המחלקה.</p> <p>לחיצה על הכותרת בראש הדף (בתמונה המוצגת, "C# Documentation") תוביל בחזרה אל הדף הראשי מכל דף בו תהיו.</p> <h2>חיפוש ב- API</h2> <p>בכל הדפים ב- API מופיע בצד שמאל התפריט הבא:</p> <p><img alt="sidebar" src="../img/Api/sidebar.png"></p> <p>בתפריט מופיעים שמות המחלקות במשחק, כאשר לחיצה על מחלקה מקשרת ישירות לדף שלה ב- API.</p> <p>לחיצה על Info Pages תפתח תת-תפריט של קבצי הסבר נוספים למשחק. כאן תוכלו למצוא את חוקי המשחק, ספריות מותרות לשימוש, הנחיות ליבוא קבצים ודפי מידע נוספים.</p> <p>הקלדה בתיבת החיפוש תשנה את תוכן התפריט, כך שנוסף על מחלקות המתאימות למילות החיפוש, יוצגו גם קישורים לפונקציות או ולמשתנים רלוונטיים.</p> <p>למשל:</p> <p><img alt="search" src="../img/Api/search.png"></p>
